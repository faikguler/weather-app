**The Hustler Collective** is a modern, responsive web application developed using **Bootstrap 4.5**.  
It integrates multiple external APIs into a single platform, allowing users to access **real-time weather data, country information, and global news** from one user-friendly interface.

The goal of this project is to eliminate the need for visiting multiple websites by aggregating essential data sources into one clean and efficient web app.
